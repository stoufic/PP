Build a full-stack web application called Kura.

Kura is a youth soccer transfer and exposure platform for players under 18. It helps players and parents create profiles, upload stats, videos, and updates, and gives verified coaches a clean way to discover talent, filter players, save watchlists, and request contact access. The platform should feel fast, modern, trustworthy, and premium.

Important product rules:
- Players are minors, so privacy and parent control are core features.
- Parents manually create and manage player profiles.
- Players and parents pay a registration fee plus a monthly fee per player.
- Coaches pay a registration fee.
- Coaches must upload their coaching license for manual verification before they can access protected player contact details.
- Player contact information must never be publicly visible.
- Contact info is visible only to verified coaches.
- Parents should have full control over their child’s profile, edits, and uploads.
- The platform should be simple, direct, and polished, not overloaded with unnecessary features.

Goal:
Create a minimal but complete full-stack MVP with a beautiful front end, functional backend, authentication, database, payments-ready structure, file uploads, role-based access control, and an admin verification flow.

Tech stack:
- Frontend: Next.js + TypeScript + Tailwind CSS
- Backend: Next.js server actions or API routes
- Database: PostgreSQL with Prisma ORM
- Auth: Clerk or NextAuth, but role-based and production-structured
- Storage: UploadThing, S3, or equivalent for player photos, videos, and coach license uploads
- Payments: Stripe integration structure for registration and monthly subscriptions
- UI: modern, premium, gradient-based, smooth hover effects, subtle animation, responsive, clean spacing
- Design style: fast, sporty, sleek, trustworthy, premium startup feel
- Keep functionality focused. Do not bloat with extra dashboards or unnecessary complex modules.

Create these roles:
1. Guest
2. Parent
3. Coach

6. Coach dashboard

  - approve or reject coaches

  - post/edit/delete news articles
  - monitor subscriptions at a simple level
