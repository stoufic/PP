Projects

Stock Predictor

In this project, I developed an advanced stock price predictor using Python, leveraging the yfinance library for historical stock data acquisition and scikit-learn for its extensive machine learning functionalities. The application allows for user-defined stock symbol input, employing sophisticated data preprocessing and feature engineering to optimize the input for the RandomForestRegressor model. With a focus on enhancing the machine learning foundation, I iterated on model selection, parameter tuning, and performance evaluation to ensure accurate and robust stock price predictions.



Automated News Email

I've made a Python email bot that smartly aggregates and condenses daily news from Google News, Bing News, and Twitter for users, based on their specific interests indicated by the notifications they've activated for certain followers. By ensuring that the bot curates content only from followed entities with active notifications, it delivers a highly personalized and relevant news digest directly to users' inboxes every evening at 10 pm, making staying informed both effortless and efficient. This project not only highlights my adeptness in API integration and Python programming but also underscores my commitment to creating user-centric solutions.



Youtube NoteTaker Chrome Extension

I developed "YouTube NoteTaker," a Chrome extension designed to enhance the learning experience on YouTube. This tool allows users to mark specific timestamps during a video to create checkpoints, where they can add detailed notes for later reference. It's particularly aimed at aiding students in their study sessions by enabling digital annotations directly within the YouTube interface. These checkpoints and notes are persistently saved within the extension, allowing users to return to them or navigate between different videos without losing their data. Completing this project honed my skills in JavaScript, Chrome API utilization, and UI/UX design, further solidifying my ability to create practical, user-focused applications.
